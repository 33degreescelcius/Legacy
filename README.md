## Legacy UI Library

**Legacy** is a Roblox scripting project focused on developing internal tools and systems for a private script environment.

This repository contains the **UI Library component** of Legacy.
It is a **private framework** intended solely for use within Legacy and by its developer (**33degreescelcius**).
This library is not distributed for public use, reuse, or integration in external projects.
